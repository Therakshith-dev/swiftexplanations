Import

import CoreLocation

What It Is

CLLocationCoordinate2D is a struct from Core Location. It represents a single geographic point using latitude and longitude.

Under The Hood

public struct CLLocationCoordinate2D { public var latitude: CLLocationDegrees public var longitude: CLLocationDegrees }
typealias CLLocationDegrees = Double

It is simply two Double values grouped together. No GPS logic exists inside this struct.

What It Does NOT Do

It does not fetch location. It does not request permission. It does not talk to hardware.

Who Gets The Location?

CLLocationManager communicates with the system. It returns a CLLocation object, which contains:

location.coordinate

Example

let coordinate = CLLocationCoordinate2D( latitude: 12.9716, longitude: 77.5946 )
