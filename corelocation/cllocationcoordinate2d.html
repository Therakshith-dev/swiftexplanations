Import Required

import CoreLocation

What It Is

CLLocationCoordinate2D is a struct from Apple's Core Location framework. It represents a single geographic point on Earth using latitude and longitude.

Under The Hood

Internally, it is defined roughly like this: 

public struct CLLocationCoordinate2D {
    public var latitude: CLLocationDegrees
    public var longitude: CLLocationDegrees
}

And: 

public typealias CLLocationDegrees = Double

This means it is simply two Double values grouped together. There is no GPS logic inside this struct.

What It Does NOT Do

  • It does not fetch location data.
  • It does not request permissions.
  • It does not communicate with GPS hardware.

It is purely a data container for coordinates.

Who Actually Gets The Location?

Location data is retrieved using: 

CLLocationManager

That returns a CLLocation object, which contains: 

location.coordinate

Example

import CoreLocation

let coordinate = CLLocationCoordinate2D(
    latitude: 12.9716,
    longitude: 77.5946
)

print(coordinate.latitude)
print(coordinate.longitude)
